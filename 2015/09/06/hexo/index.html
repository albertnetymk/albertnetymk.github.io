Migrating to Github Pages with Hexo

Sep 6 2015

Given that Scriptogram is about to shutdown soon, I have to find a new home for my blog. Github is generous enough to provide free hosting for static sites, but Jekyll just feels so complicated to use. Fortunately, there’s this static site generator list, and Hexo seems fairly popular.

It’s rather straightforward to get it up and running, with some minor modifications from the format used in Scriptogram.

  • Title -> title
  • Date -> date
  • --- to separate the header and the body
  • Move posts with Type: page to its own folder

It’s less convenient compared with the way how Scriptogram works, but bearable.
